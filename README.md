# VRPlay(built in ReactVR)
A simple demo video player built in ReactVR 1.0

## Intallation
```command
$ git clone git@github.com:amazingandyyy/react-vr.git
$ yarn
$ npm start
```
open browser in `http://localhost:8081/vr/`
